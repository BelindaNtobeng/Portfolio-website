# Belinda Ntobeng's Portfolio Website

## Full-stack portfolio website built with HTML, CSS, JavaScript, and Node.js featuring REST API integration and backend form handling.

## Overview
This project is a full-stack portfolio website developed to present my software engineering projects, technical skills, and professional background in a structured and interactive format.

The application consists of a responsive frontend built with HTML, CSS, and vanilla JavaScript, and a backend powered by Node.js and Express. The backend handles contact form submissions, server-side validation, and API endpoints for dynamic content rendering.

The primary objective of this project is to demonstrate practical full-stack development skills, including frontend design, REST API integration, backend architecture, and deployment.

## Problem Statement
Recruiters need more than a resume to evaluate technical ability. This platform provides direct access to my projects, GitHub repositories, and technical stack in a structured and interactive format.

## Tech Stack
FRONTEND: HTML, CSS, JavaScript
BACKEND: Node.js, Express.js
DATABASE: MySQL

## Features
- Responsive layout for desktop and mobile
- Project showcase with GitHub links
- Backend-powered contact form with validation
- REST API integration
- Secure CV download endpoint serving static PDF asset
- Skills section categorized by domain
